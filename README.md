# Sacchan Token (SAC)

Sacchan Token is a premium Web3 landing page inspired by the true story of Sacchan, the Osaka dog who became a local legend after a 1992 episode of `Knight Scoop` revealed that an entire town was collectively feeding and caring for him.

The site translates that story into a modern crypto brand centered on decentralized participation, shared ownership, transparent incentives, and organic growth.

## Brand Story

Sacchan was a famously chubby dog living in Osaka, Japan. His owner noticed that he kept gaining weight and could not understand why. A television investigation revealed the answer: Sacchan walked a route through town every day, and dozens of people fed him along the way.

Shop owners, station workers, neighbors, residents, and strangers all contributed small acts of generosity. No single person was responsible. Instead, an entire decentralized network of people collectively supported Sacchan.

That is the philosophy behind SAC.

- Next.js

## Project Structure

- `app/` - routes, global layout, and styling
- `components/` - reusable UI sections
- `lib/` - shared content and copy
- `public/logo.JPG` - Sacchan mascot and token logo image used in the header, hero panel, and community section

## Setup

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## Notes

This build uses future-facing tokenomics placeholders and placeholder CTA destinations so the brand and layout can ship before the token mechanics are finalized.
